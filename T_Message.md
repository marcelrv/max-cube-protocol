## The t Command

The `t:` command deletes a room or a device from the Cube. 
The cube  acknowledge the the processing of a command with an `A:` message. 


```
t:01,1,Dx1U
A:
```

Details
```
Position   Information
===================================================
t:        Command
01        Group/Room?
1         Force yes/no
Dx1U      Group/Room
```
